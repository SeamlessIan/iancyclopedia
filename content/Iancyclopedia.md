Ian's personal wiki. Originally a self-hosted MediaWiki site, abandoned after a spam bot incident. Revived in 2026 as a static site built with [Quartz 5](https://quartz.jzhao.xyz/), hosted on GitHub Pages.

**Live at:** https://iancyclopedia.com  
**Repo:** https://github.com/SeamlessIan/iancyclopedia  
**Stack:** Quartz 5, GitHub Actions (auto-deploys on push to main), custom domain via Cloudflare

Content recovered from a 2019 SQL backup, 196 articles as of July 2026.
