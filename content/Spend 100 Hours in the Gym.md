## Fitness Plan Part 1: Starting Strength
The purpose of this part of the fitness plan is to be stronger, have more energy, look better, and be more fit as I age. 

Starting Strength is a program originally designed by Mark Rippletoe for beginning weightlifting. The program consists of two workouts (called A and B). Each workout consists of 3 lifts. For each lift, do 3 sets of 5 reps, then move on to the next lift (except the deadlift, which is 1 set of 5 reps). Alternate between the two workouts each session, and lift on non-consecutive days. Using this system, you should be able to make consistent gains for a period of time. Once you plateau, you can consider moving to a more complicated system.
### Getting Started
- [Watch/read tutorials for proper form for each exercise](http://forum.bodybuilding.com/showthread.php?t=998224)
- Take "before" progress pics
- Schedule monthly progress pics
- Get gym membership
- Record baseline strength for each exercise
- Start lifting on alternating days! Make sure to track your progress, like in a spreadsheet or document.
- Increase weight marginally every workout
- When you plateau, find a new program

### Workout A
- 3x5 Squat
- 3x5 Bench Press
- 1x5 Deadlift

### Workout B
- 3x5 Squat
- 3x5 Standing military press
- 3x5 Pendlay Rows
